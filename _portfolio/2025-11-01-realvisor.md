RealVisor is an AI-powered real estate analytics platform built using Python, Streamlit, and Machine Learning. It predicts apartment prices, analyzes housing trends, and recommends properties based on data insights.

### Key Features
* **Predictive Pipeline:** Utilizes XGBoost and Random Forest Regressors to estimate property valuations based on complex spatial and architectural features.
* **Interactive Dashboard:** A production-ready Streamlit interface featuring market trend visualizations, property comparison tools, and automated EMI calculations.
* **Tech Stack:** Python, Streamlit, Scikit-learn, Pandas, NumPy, Plotly, Joblib.
